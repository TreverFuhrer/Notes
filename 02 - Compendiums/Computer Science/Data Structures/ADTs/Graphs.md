## Overview

| Operations | Description                   |
| ---------- | ----------------------------- |
| `push(x)`  | Adds an element to the stack. |
| `pop()`    | Removes the top element.      |

### Components
- **Vertices (Nodes):** Represent entities in the graph. For example, cities in a map, users in a social network, etc.
- **Edges (Connections):** Represent relationships or connections between vertices. For example:
	- In an airline map, edges represent flights between cities.
	- In a social network, edges represent friendships or followings.

#### Properties of Edges:
- **Directed vs. Undirected:**
    - **Directed:** Edges have a direction (e.g., Twitter follow relationships).
    - **Undirected:** Edges have no direction (e.g., Facebook friendships).
- **Weighted vs. Unweighted:**
    - **Weighted:** Edges have a weight or cost (e.g., distance between cities, bandwidth of a connection).
    - **Unweighted:** All edges are treated equally.
